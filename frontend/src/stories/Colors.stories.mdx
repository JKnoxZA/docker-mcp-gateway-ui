## Usage Guidelines

### Accessibility

All color combinations meet WCAG 2.1 AA standards for color contrast:

- **Normal text**: Minimum 4.5:1 contrast ratio
- **Large text**: Minimum 3:1 contrast ratio
- **UI components**: Minimum 3:1 contrast ratio

### Semantic Meaning

Use colors consistently to convey meaning:

- **Green**: Success, positive actions, running/active states
- **Red**: Errors, destructive actions, stopped/failed states
- **Yellow**: Warnings, caution, pending states
- **Blue**: Information, primary actions, links
- **Gray**: Neutral content, secondary actions

### Examples

```css
/* Success states */
.status-running { @apply bg-green-100 text-green-800; }
.button-success { @apply bg-green-600 hover:bg-green-700; }

/* Error states */
.status-error { @apply bg-red-100 text-red-800; }
.button-destructive { @apply bg-red-600 hover:bg-red-700; }

/* Warning states */
.status-warning { @apply bg-yellow-100 text-yellow-800; }

/* Info states */
.status-info { @apply bg-blue-100 text-blue-800; }
```
